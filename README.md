#Bomberman Project

Bomberman är ett multiplayer-spel där spelare styr sina karaktärer på en begränsad bana. Målet är att eliminera motståndare genom att placera bomber och samla power-ups.

##Kontroller

*Spelare 1:

*Rörelse: W (upp), A (vänster), S (ner), D (höger)
*Placera bomb: Space

*Spelare 2:

*Rörelse: Piltangenter
*Placera bomb: End
##Funktionalitet
*Karaktärer kan röra sig med WASD eller piltangenter.

*Bomber kan placeras och exploderar efter en kort tid.

*Block (bricks) kan förstöras för att nå motståndare.

*Vissa block kan droppa power-ups som förbättrar spelarens förmågor.

*Bomber kan skjutas eller skuffas för strategiska spelmoment.
