Bomberman Project

Bomberman är ett multiplayer-spel där spelare styr sina karaktärer på en begränsad bana. Målet är att eliminera de andra spelarna genom att placera bomber och samla power-ups.

Kontroller
Spelare 1:
Rörelse: W (upp), A (vänster), S (ner), D (höger)
Placera bomb: Space
Spelare 2:
Rörelse: Piltangenter
Placera bomb: End
Funktioner
Rörelse av karaktärer med WASD eller piltangenter
Placera bomber som exploderar efter kort tid
Förstöra block (bricks) för att nå motståndare
Block kan droppa power-ups
Skuffa (push) bomber
